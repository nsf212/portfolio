Developpement WEB

Création d'un site web pour le restaurant "18ème"

L’objectif de ce projet est de concevoir et développer un site web vitrine pour le restaurant "18ème" afin de présenter ses services et son menu.

  • Présentation du restaurant et de son concept
  • Affichage du menu avec les différents plats proposés
  • Formulaire de réservation en ligne
  • Section contact avec informations et localisation

Le développement du site se déroulera en plusieurs étapes :

  • Conception des maquettes et architecture du site
  • Développement en HTML, CSS et JavaScript
  • Ajout d'interactivité avec JavaScript
  • Optimisation de l'affichage pour le responsive design
